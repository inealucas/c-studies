## Key concepts
- `scanf` reads formatted input from the user.
- `fgets` can read full strings with spaces.
- `scanf("%s")` stops reading at spaces.
- The `\n` character stays in the input buffer after pressing Enter.
- Adding a space before `%c` ignores whitespace and `\n`.
- Strings in C end with the null character `'\0'`.
- Uninitialized variables can contain garbage values.
